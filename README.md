# 🍎 おちものぱずる (Ochimon Puzzle)

物理演算エンジン Matter.js を使用して作成された、シンプルな落ち物系パズルゲームです。

同じサイズのフルーツを重ねて合体させ、どんどん大きなフルーツに進化させてハイスコアを目指します。

## 🕹️ 遊び方 (How to Play)

1.  **フルーツを落とす**: 画面上部をタップまたはクリックすることで、次に落ちてくるフルーツを指定した位置に落下させます。
2.  **合体**: 落下したフルーツが、同じ種類のフルーツと接触すると合体し、**次のサイズのフルーツ**に進化します。
3.  **ハイスコア**: 合体を繰り返して大きなフルーツを作ることで、高得点が得られます。
4.  **ゲームオーバー**: フルーツが積み上がり、画面上部の**成長ライン**（赤い点線）を超えた状態で一定時間静止するとゲームオーバーになります。

## 🛠️ 技術スタック (Tech Stack)

このプロジェクトは、単一のHTMLファイル内で完結するように設計されています。

* **HTML5 / CSS3**: 基本的な構造とスタイル設定。
* **JavaScript (ES6+)**: ゲームロジックの実装。
* **Matter.js (v0.19.0)**: 高度な物理演算（落下、衝突、反発）を実現するためのエンジンとして使用。
* **Canvas API**: フルーツ、スコア表示、パーティクルエフェクトなど、すべてのゲームグラフィックのレンダリングに使用。

### ✨ 特徴

* モバイルデバイスでの動作を考慮し、タッチ操作（スワイプでの位置調整、タップでの落下）に対応しています。
* 一番小さいフルーツの色を**青**、その次を**水色**に変更するなど、独自のデザインを採用しています。
